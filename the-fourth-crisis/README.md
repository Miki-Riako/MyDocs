## The Fourth Crisis Farming Pack

第四天灾种田包，旨在加强各位第四天灾的种田体验。
目前自用，持续更新中。
大部分功能需要启用第四天灾才能生效，不启用可以当作优化了种田体验的偏原版内容进行游戏。
